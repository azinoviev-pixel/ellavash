# INVENTORY_SCALE_v1 — инвентаризация нереалистичных масштабов

Дата: 28.05.2026
Контекст: Элен — одна, без сотрудников. Производство стартует с 5-10 кг/день, потолок реалистичного роста за 3 месяца — 20-30 кг/день. НЕ тонны, НЕ сетевой ритейл в первые 6 мес.

Цель документа: выявить все цитаты по 10 ключевым файлам, где цифры/масштабы/каналы звучат нереалистично высоко для одинокого стартапа на старте, и предложить замены.
