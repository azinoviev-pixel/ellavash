6. Каталанский язык — есть ли смысл?

Аргументы ЗА

  • Льорет-де-Мар = Каталония, локальная аутентичность повышает доверие
  • В каталанском SERP по pa àrab, pa de pita доминируют рецепт-блоги (receptes.cat, tombatruites.cat, bruixesalacuina.blogspot.com) — почти нет коммерческих игроков
  • Каталоговая идентичность («producte de Catalunya», «artesà de Lloret») — сильный moat против Mercadona/Glovo

Аргументы ПРОТИВ

  • Объёмы каталанских запросов примерно в 8–15 раз ниже испанских (по аналогии с другими нишами — стандартное соотношение для Cataluña)
  • Слово lavash в каталанском не имеет устойчивой формы: используется как заимствование lavash (Viquipèdia использует это слово). Lavaix — народная транслитерация, низкий volume
  • 92% каталонских интернет-пользователей читают на испанском без проблем

Рекомендация

Делать каталанскую версию частично, во второй фазе, и только для 4 страниц:
- главная (/ca/)
- история / heritage (/ca/que-es-el-lavash/ → /ca/que-es-el-pa-lavash/)
- локальный лендинг Lloret (/ca/forn-de-lloret-de-mar/)
- B2B HORECA Cataluña (/ca/distribuidor-horeca-catalunya/)

Это закроет «локальную аутентичность» и не потребует переводить весь сайт.

8. Сводка (300 слов)

Реальные данные Ahrefs API получить не удалось — подписка возвращает «Insufficient plan» на все запросы Keywords Explorer и SERP Overview. Все объёмы и KD в документе — оценочные, основанные на ручном анализе SERP Google.es через web-поиск. После апгрейда подписки нужен повторный прогон для калибровки цифр.

Топ-10 keywords с лучшим ROI для El Lavash (в порядке приоритета): lavash, pan lavash, donde comprar lavash en Barcelona, comprar lavash, comprar lavash online, diferencia lavash y pita, como hacer lavash, comprar pan árabe por mayor, lavash congelado, panadería Lloret de Mar.

Главный SEO-вывод: по ядру lavash и pan lavash ТОП-10 Google.es состоит из Wikipedia, рецепт-блогов и Alibaba — ни одного коммерческого бренда лаваша. Это уникальное окно: одна хорошая pillar-страница (3 500+ слов с историей, рецептом, видео, где купить) с органичной внутренней перелинковкой на e-commerce может занять ТОП-5 за 3–6 месяцев. Локальные запросы (donde comprar lavash en Barcelona, panadería Lloret de Mar) ещё пустее — там нет даже Wikipedia, можно встать в ТОП-3 одним лендингом + Google My Business.

По каталанскому: делать смысл, но ограниченно — только 4 страницы (главная, история, Lloret-лендинг, HORECA-дистрибьюция). Объёмы в каталанском в 8–15 раз ниже испанских, но конкуренция почти нулевая, и это даёт сильный сигнал локальной аутентичности против Mercadona и сетевого опта.

Какой контент создавать первым: pillar-guide /que-es-el-lavash/ (захватывает главное ядро) + локальный лендинг /comprar-lavash-en-barcelona/ (захватывает чистую транзакцию без конкурентов) + категорийная страница /comprar-lavash/. Эти три страницы + правильный GMB в Lloret должны принести первый органический трафик за 6–8 недель и закрыть 60% коммерческого SEO-потенциала El Lavash на испанском рынке.
