## 3. Эмодзи запрещены

Запрещено использовать эмодзи в любых файлах проекта, в названиях, в комментариях, в интерфейсах, в чате, в коде.

**Чем заменять:**
- **HTML:** только inline SVG-иконки (рисовать самому или брать из Lucide/Heroicons/Phosphor/Tabler как SVG-разметку)
- **Markdown / CSV / TXT:** текстовые метки в квадратных скобках: `[!]`, `[STOP]`, `[CRIT]`, `[OK]`, `[v]`, `[i]`, `[X]`
- **React:** `<AlertTriangle />` и т.д. как React-компоненты с inline SVG
- **Дизайн:** через Adobe Firefly, Canva, Figma, Illustrator

**Что не считается эмодзи** и допустимо:
- Типографика: `•`, `→`, `←`, `↑`, `↓`, `▸`, `◆`, `■`, `□`, `§`, `¶`
- Математика: `+`, `−`, `×`, `÷`, `±`, `≈`, `≠`, `≤`, `≥`
- Кавычки/тире: `«»`, `„"`, `—`, `–`

## 4. Мобильная-обязательность

**Каждый HTML-документ, который попадает на ellavash.es или передаётся клиентке — обязан корректно работать на мобильном.**

Перед сдачей любого HTML файла **проверять:**
1. `<meta name="viewport" content="width=device-width, initial-scale=1.0">` стоит в `<head>`
2. Все сетки/карточки переходят в одну колонку при `max-width: 800px` (или меньше)
3. Шрифт base ≥15px на мобильном, заголовки через `clamp()` или явные мобильные размеры
4. Нет горизонтального скролла на 375px вьюпорте
5. Тапы (кнопки/ссылки) ≥ 44×44 px
6. Таблицы — `overflow-x: auto` на контейнере или конвертация в карточки на узких экранах
7. Картинки — `max-width: 100%; height: auto;`
8. Footer/header не обрезаются на iPhone (учитывать safe-area)

**Контроль перед деплоем:**
- Открыть страницу в режиме DevTools (375px iPhone, 412px Android)
- Скриншоты mobile-view сохранять рядом с файлом (опционально, если объём большой)

Никакой публикации без mobile-проверки.

## 6. Палитра проекта (из логотипа)

Извлечено пиксельным анализом `logo_web.png`:

| Цвет | HEX | RGB | Использование |
|---|---|---|---|
| Кофейный (deep brown) | `#6F3E22` | rgb(111, 62, 34) | Основа фона / акценты |
| Бумажный (parchment) | `#FFF3D9` | rgb(255, 243, 217) | Текст на тёмном / светлый фон |
| Горчичный (saffron) | `#F8D202` | rgb(248, 210, 2) | Brand primary / call-to-action |
| Томатный (tomato red) | `#E61E28` | rgb(230, 30, 40) | Brand secondary / alerts |

Палитра для тёмной темы (новый базовый стиль):
```css
--bg: #2A1810;       /* фон страницы — посветлее чем v1 (#1A130E), но всё ещё тёмный */
--bg-soft: #3A2417;  /* поднятые поверхности */
--bg-card: #4A2E1D;  /* карточки */
--ink: #FFF3D9;      /* основной текст */
--ink-soft: #C8AB7E; /* приглушённый текст */
--ink-dim: #8A6F50;  /* подписи */
--line: #5A3923;     /* границы */
--brand: #F8D202;    /* основной brand */
--brand-soft: #C8A902; /* затемнённый brand */
--accent: #E61E28;   /* акцент / алерт */
--coffee: #6F3E22;   /* кофейный mid-tone */
```

Светлая тема (если когда-то понадобится для отдельной страницы):
```css
--bg: #FFF3D9;
--ink: #2A1810;
--brand: #6F3E22;
--accent: #E61E28;
```
