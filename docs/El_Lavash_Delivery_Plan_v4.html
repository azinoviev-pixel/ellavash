Чего избегать на старте

Just Eat: активационная плата до €699 (по другим источникам — €299 + €100 депозит). Не подписывать без попытки выторговать waiver. Glovo Premium / Sponsored placements — сначала органика, потом платное продвижение. Bolt Food в Льорете не подтверждён — не торопиться. Deliveroo, Yango Eats, La Nevera Roja, MyMenú в Испании 2026 либо не работают, либо поглощены.

Чек-лист

Что готовить перед регистрацией

Большинство платформ требуют один и тот же пакет. Подготовить один раз — отправлять везде.

Юр. документы

  • NIF / CIF компании
  • Alta IAE (деятельность Каталония)
  • Registro Sanitario (RGSEAA)
  • HACCP / план самоконтроля
  • Страховка ответственности продукта
  • PRL básico (охрана труда)
  • Опц.: Halal / Kosher / Bio

Финансы

  • Банковский счёт IBAN ES
  • SEPA-мандат
  • Email для invoicing
  • Прайс-лист по позициям
  • Минимальные заказы по каналам
  • Калькуляция себестоимости
  • Учётная политика НДС

Контент

  • Логотип PNG + SVG
  • Каталог продукции (имя, аллергены, вес)
  • Фото каждой позиции ≥ 1200×1200
  • Фото производства / печки
  • Описание бренда RU + ES + CA
  • Таглайн 15-30 знаков
  • Часы работы по каналам

Логистика

  • Адрес производства / pickup
  • Радиус доставки (Льорет 30 км)
  • Cold-chain для дальних заказов
  • Стандарт упаковки горячая / охл. / замороженная
  • SEUR Frío аккаунт (для всей ES)
  • Контракт Uber Direct / Stuart
  • Контактный телефон WhatsApp
